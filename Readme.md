# NIAD-TO: Neural-Implicit and Adjoint-Driven Topology Optimization

This project provides an end-to-end topology optimization framework that combines neural implicit density representation (PyTorch) with discrete adjoint sensitivities (FEniCS/dolfin-adjoint). The hybrid gradient flow enables scalable optimization for PDE-constrained problems using standard PyTorch optimizers.
